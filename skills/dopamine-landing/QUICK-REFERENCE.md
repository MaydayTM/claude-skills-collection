# Dopamine Landing Skill - Quick Reference

## The 5-Section Formula

### 1️⃣ VISUAL IMPACT (15-25 words max)
- ✅ Looping video or animation
- ✅ Bold, large headline (48-72px)
- ✅ Minimal text
- ❌ No walls of text

### 2️⃣ CURIOSITY LOOP (30-50 words max)
- ✅ Ask an unanswered question
- ✅ Amplify a problem
- ✅ Create contrast (familiar vs. shocking)
- ❌ Don't give away the answer yet

### 3️⃣ MYSTERY BUILDING (5 words per scroll point)
- ✅ Scrollytelling animations
- ✅ Progressive reveals
- ✅ One visual metaphor that transforms
- ❌ No static dumps of information

### 4️⃣ THE SOLUTION (75-150 words max)
- ✅ Deliver the surprising answer
- ✅ Back it up with data/visuals
- ✅ Close the curiosity loop
- ❌ Don't make it too simple/obvious

### 5️⃣ TRUST & CTA (150-250 words max)
- ✅ Show real human faces
- ✅ Specific testimonials
- ✅ One clear CTA
- ❌ No multiple competing CTAs

## Animation Triggers

| Section | Trigger Type | When |
|---------|--------------|------|
| Section 1 | Auto-play | On page load |
| Section 2 | Fade-in | On scroll (60% visible) |
| Section 3 | Scroll-triggered | Progressive (multiple points) |
| Section 4 | Reveal | On scroll (60% visible) |
| Section 5 | Hover effects | On CTA interaction |

---

## Typography Scale

```
H1 (Hero):        48-72px, bold
H2 (Section):     32-48px, bold
H3 (Subsection):  24-32px, medium
Body:             16-18px, regular
Captions:         12-14px, light
```

---

## Content Checklist

Before launching, verify:

- [ ] Section 1 has movement/animation
- [ ] Section 2 opens a clear question
- [ ] Section 3 delays the answer with scrollytelling
- [ ] Section 4 provides surprising resolution
- [ ] Section 5 shows real faces/testimonials
- [ ] Total word count < 500 words
- [ ] Visual hierarchy is clear
- [ ] One primary CTA only
- [ ] Mobile responsive animations

---

## Common Mistakes

| ❌ Don't | ✅ Do |
|----------|-------|
| Static hero image | Looping video/animation |
| Answer the question early | Build mystery, delay resolution |
| Too much text everywhere | Minimal text, maximum visuals |
| Generic stock photos | Real team/customer photos |
| Multiple CTAs | One clear, prominent CTA |
| Explaining everything | Create gaps the brain wants to close |

---

## Success Metrics

Track these to measure effectiveness:

- **Bounce Rate:** Target <40%
- **Scroll Depth:** Target >70% reach Section 4
- **Time on Page:** Target >60 seconds
- **Conversion Rate:** Target 2-5x baseline

---

**Remember:** The goal is to trigger dopamine at each stage. If a section doesn't create anticipation, curiosity, or satisfaction, revise it.
