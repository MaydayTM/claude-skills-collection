## Typography (Lettertype Selectie)

### Primair Font (Headings)
Gebruikt voor titels, koppen, logo.

**Stijl opties:**

| Type | Uitstraling | Voorbeelden |
|------|-------------|-------------|
| **Sans-serif** | Modern, clean, tech | Inter, Satoshi, DM Sans |
| **Serif** | Traditioneel, premium, editorial | Playfair Display, Lora, Merriweather |
| **Display** | Bold, opvallend, creatief | Clash Display, Cabinet Grotesk |
| **Mono** | Tech, code, minimal | JetBrains Mono, Space Mono |

**Jouw keuze:** _______________
**Waarom:** _______________

### Secundair Font (Body)
Gebruikt voor lopende tekst, paragrafen.

**Regel:** Kies een goed leesbaar font, vaak sans-serif.

**Populaire combinaties:**
- Heading: Clash Display → Body: Inter
- Heading: Playfair Display → Body: Source Sans Pro
- Heading: Cabinet Grotesk → Body: DM Sans
- Heading: Space Grotesk → Body: Inter

## Visuele Stijl Richting

### Kies je Stijl (1 primair, 1 secundair)

| Stijl | Kenmerken | Wanneer kiezen |
|-------|-----------|----------------|
| **Minimalistisch** | Veel witruimte, weinig kleuren, clean | Premium, tech, focused |
| **Bold/Maximalistisch** | Grote letters, felle kleuren, opvallend | Creative, entertainment |
| **Editorial** | Magazine-achtig, serif fonts, foto's | Media, lifestyle, fashion |
| **Tech/Modern** | Gradiënten, geometric shapes, sans-serif | SaaS, startups, tech |
| **Organic/Warm** | Zachte vormen, aardetinten, handgetekend | Wellness, coaching, persoonlijk |
| **Corporate/Professional** | Strak, blauw/grijs, traditioneel | B2B, finance, consultancy |
