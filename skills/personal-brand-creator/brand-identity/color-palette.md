## De Psychologie van Kleur

| Kleur | Emotie/Associatie | Geschikt voor |
|-------|-------------------|---------------|
| **Blauw** | Vertrouwen, professionaliteit, rust | Tech, finance, healthcare, corporate |
| **Navy/Donkerblauw** | Autoriteit, expertise, luxe | Premium diensten, consultancy |
| **Groen** | Groei, gezondheid, natuur, geld | Wellness, duurzaamheid, finance |
| **Teal** | Modern, kalmerend, uniek | Tech, health, creative |
| **Rood** | Energie, urgentie, passie | Entertainment, food, sales |
| **Oranje** | Vriendelijk, energiek, toegankelijk | Creative, youth, startups |
| **Geel** | Optimisme, warmte, aandacht | Creative, coaching, education |
| **Paars** | Creativiteit, luxe, wijsheid | Creative, luxury, spiritueel |
| **Roze** | Modern, speels, empathisch | Fashion, beauty, lifestyle |
| **Zwart** | Luxe, elegantie, kracht | Premium, fashion, minimalist |
| **Wit** | Zuiverheid, eenvoud, ruimte | Minimalist, tech, healthcare |
| **Grijs** | Neutraal, professioneel, tijdloos | Corporate, tech, professional |

## Chris Do's Voorbeeld: The Futur

| Element | Kleur | Hex | Waarom |
|---------|-------|-----|--------|
| Primary | Elektrisch Blauw | #0066FF | Modern, toekomstgericht |
| Secondary | Zwart | #000000 | Premium, contrast |
| Tertiary | Wit | #FFFFFF | Clean, ruimte |

**Waarom het werkt:**
- Minimalistisch = focus op inhoud
- Blauw = vertrouwen + professionaliteit
- Hoog contrast = moderne, designer aesthetic
