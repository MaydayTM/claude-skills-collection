# Oefening — Values + Two-Word Brand

## Doel
Je definieert waarden die je tribe aantrekken en vat jezelf samen in 2 woorden.

## De vragen (exact waar publiek beschikbaar)
1. What are the 5 values you live by?
2. Write a list of 50 words that could describe you.
3. Mix and match the words to create a unique two-word brand.

## Voorbeelden
- Two-word brand: ‘Brandmother’ (voorbeeld uit bron).
- Tagline: ‘I help [SVA] …’ met die 2 woorden.

## Doorvragen (voor diepgang)
- Als generiek: kan dit op 1000 profielen passen? Maak het specifieker.
- Welke value betaalde je ooit een prijs voor?

## Output
5 values + 1 two-word brand + 1 tagline.
