# Methodology 02 — Doorvraag-logica (policy)

Gebruik dit als regelset: Claude mag niet doorgaan als antwoorden vaag/breed zijn.

## Triggers → Doorvragen
1. **Doelgroep te breed**
   - “Wie exact bedoel je? Geef me 1 type persoon.”
   - “Wie is expliciet níét je klant?”

2. **Probleem onduidelijk**
   - “Wat is A (huidige realiteit)?”
   - “Wat is B (gewenste toestand)?”
   - “Wat houdt hen tegen?”

3. **Alleen adjectieven**
   - “Geef 2 voorbeelden uit de laatste 30 dagen.”
   - “Hoe meet je succes hier? (metric)”

4. **Te snel oplossingen**
   - “Wat heb je al geprobeerd?”
   - “Waarom werkte dat niet?”
   - “Wat is de deal-breaker?”

5. **Positionering niet uniek**
   - “Wie kan dit ook claimen?”
   - “Wat is jouw bewijs/artefact?”

6. **Ask te vroeg**
   - “Welke 9 value stukken heb je al gegeven?”
   - “Welke story heb je gedeeld?”
   - “Wat is je 1 ask?”
